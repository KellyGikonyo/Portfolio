# Portfolio Project Structure

This portfolio project is designed to showcase a variety of web development skills and projects. Below is an overview of the project structure:

## Project Structure

```
Portfolio/
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── script.js
├── images/
│   ├── project1.jpg
│   ├── project2.png
│   └── project3.gif
└── files/
    └── resume.pdf
```

### Description of Directories and Files

- **index.html:** This is the main HTML file for the portfolio. It contains the layout and structure of the webpage.
- **css/:** This directory contains all the stylesheet files, including `styles.css`, which is linked to the index.html and controls the overall styling of the website.
- **js/:** This folder contains JavaScript files such as `script.js`, which adds interactivity and functionality to the portfolio.
- **images/:** This directory holds all images used in the portfolio, including project screenshots and logos.
- **files/:** This folder contains supplementary files like resumes and PDFs that may be linked within the portfolio.
